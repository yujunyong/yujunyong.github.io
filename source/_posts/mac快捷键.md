## terminal中的快捷键
* Command + K：清屏
* Command + T：新键标签
* Command + W：关闭当前标签页
* Command + S：保存终端输出
* Command + D：水平分隔当前标签页
* Command + Shift + D：关闭当前水平分隔标签页
* Command + Shift + {：向左切换标签页
* Command + Shift + }：向右切换标签页
